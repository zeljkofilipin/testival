[Meetup](https://www.meetup.com/SoftwareTestingClub/events/53465042/) (10)

We are happy to announce Zagreb STC MeetUp #3 event that we will organize on Thursday, March 15, 2012, 5:30 PM at the Hvar conference room in Ericsson Nikola Tesla, Krapinska 45 Zagreb. Topic of this event is load testing. For one real application we will do test plan, choose appropriate test tool and do test execution of several scenarios. Also, we will distribute no. 6 issue of Software Testing Planet! In addition, we are looking for event sponsors that would provide the event with refreshments and snacks. For any info, please contact Karlo Smid at karlo(dot)smid(at)gmail(dot)com.
