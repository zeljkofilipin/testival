[Meetup](https://www.meetup.com/testival/events/243175286/)

Regular talks (10-30 minutes)

#1 Vilim Stubičan: Improving love-hate relationships between developers and testers

Take a look into some of the standard relationships between developers and testers from a developer's perspective. How they communicate, what is important for each side and what do they hate about the others. You will learn how to improve general love-hate relationship with your developers or testers through some experiences and lots of tips and tricks.

#2 Marko Kruljac: Security testing web API-s

We will talk about common test patterns and testing heuristics when doing API security testing. This will be a hands-on approach showing you some of the tools and practices that can help you when doing web API security testing.

Lightning talks (5 minutes)

#1 Željko Filipin, Freelance software engineer: Career development. Because software is not the only thing you should develop.
