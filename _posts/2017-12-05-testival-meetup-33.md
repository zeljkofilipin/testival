Regular talks (10-30 minutes)

#1 Karlo Smid: How to come up with test ideas using lateral thinking

One of the most important software tester skill is to come up with test ideas. There is a number of test techniques that help tester to do this task. In this presentation I will talk about one such test technique: lateral thinking. Lateral thinking is simply put the skill to think outside the box. This talk will also have fun part when you will get your chance to practice lateral thinking.

#2 Ana Prpić: Selenium - from 😳 to 🤔 to 😎.

This talk will feature an overview of Selenium - how it came to be, what does it do, how to use it, and a live demo.
