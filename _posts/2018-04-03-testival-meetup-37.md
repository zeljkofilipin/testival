[Meetup](https://www.meetup.com/testival/events/248535507/)

Introduction (5-10 minutes)

At the beginning of every meetup, we get to know each other. Be prepared to say a sentence or two about yourself. If you are looking for a job or hiring, this would be a good time to mention it.

Regular talks (10-30 minutes)

#1 Ana Prpić: Mobile Testing with Appium and Docker

Ana is a QA engineer at Oradian, resident WebdriverIO guru and a lover of all things coffee.

#2 Vilim Stubičan: Testing (by) documentation

How nice would it be to have documentation for every API you have? Well, if you have it by any chance, there is a way to actually validate your API with that same documentation. You'll see how you can do this with two documentation standards and almost any major language platform (you choose).

Lightning talks (5 minutes)

#1 Karlo Smid: example of providing feedback on conference abstract proposal using Mind Map.
