Zagreb STC #11 meetup held at Zip.Factory.

The meetup invitation calls for all interested developers/testers to attend a collaborative session aimed at understanding the OAuth protocol through practical testing.

Tools to be used:

- Freemind mind map tool
- OWASP ZAP proxy
- Chrome developer tools
- curl
- Manual analysis (our brains)

Pizza and refreshments were sponsored by Tentamen.
