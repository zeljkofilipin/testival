Regular talks (10-30 minutes each)

#1 Tomislav Fabeta, Degordian: Jenkins Pipelines

Jenkins Pipeline is a suite of plugins which supports implementing and integrating continuous delivery pipelines into Jenkins. Idea is to automate application delivery with steps from version control, through builds and other processes to customer. Pipeline is created through JenkinsFile which is added to project root and uses configs configured in Jenkins. Blue Ocean Jenkins plugin comes very handy when we need to create complex Pipelines giving us a nice GUI. Tomislav will describe how Pipelines can be integrated in a project and how to create a simple Pipeline using Blue Ocean plugin for Jenkins.
