Regular talks (10-30 minutes each)

#1 Filip Rakić (KING ICT) - Automated API testing

Manual API testing can be hard. There's usually a lot of data and, more often than not, covering every possible combination by hand it's not even feasible. Enter JMeter -- an open-source Java application used for automated testing. Filip will discuss the advantages and disadvantages of API test automation. In the end, you will be able to see some real-life examples of testing with JMeter.

#2 Vanja Pisačić (KING ICT) - Behavior Driven Development Test Framework

Vanja will talk about Behavior Driven Development Test Framework developed for KING ICT projects and he will discuss what components are in a framework and what advantages does this framework brings. He will show all important things from the beginning to the end on a real project such as creating feature files on TestRail, import feature files in framework, creating step definitions and page objects, parametrization, integration with Jenkins, reporting and sending notifications on email and Slack.
